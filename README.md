# Escolha Combustível

Aplicativo React Native (Expo) para ajudar motoristas a escolher entre abastecer com gasolina ou etanol, baseado no consumo do carro e nos preços informados.

## Funcionalidades
- Cadastro de carros (nome, consumo médio)
- Entrada dos preços de gasolina e etanol
- Cálculo do combustível mais vantajoso
- Interface simples e acessível

## Estrutura de Pastas
```
src/
  components/
  screens/
  services/
  assets/
  utils/
```

## Como rodar o projeto
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor Expo:
   ```bash
   npm start
   ```
3. Use o Expo Go no celular ou emulador para testar o app.

## Padrões de Código
- ESLint e Prettier configurados para padronização
- Projeto em TypeScript
